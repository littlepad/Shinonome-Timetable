りんかい線 東雲駅の時刻表（Chromeの拡張機能）  
=======

DESCRIPTION
-------
直近1時間の電車を表示します。  
週末は対応していますが、今のところ祝日は未対応。

**拡張機能の取り込み方**   
Chromeの設定メニューを開き、[ツール] → [拡張機能] を選んで拡張機能管理画面を呼び出す。  
“デベロッパー モード” のチェックボックスにチェックを入れる。  
表示された [パッケージ化されていない拡張機能を読み込みます…] をクリックする。  
ダイアログが表示されるので、publicフォルダを指定して [OK] をクリックする。
